Porsche 911 GT3 RS

Unmatched Performance and Precision

The Porsche 911 GT3 RS is engineered for unparalleled performance, boasting a 4.0L flat-six engine that delivers 520 horsepower. With precision-tuned aerodynamics and a lightweight body, this car is built for the racetrack while remaining road-legal.

SPECIFICATIONS

  • Engine: 4.0L Flat-6
  • Power: 520 hp @ 8250 rpm
  • Torque: 346 lb-ft @ 6250 rpm
  • 0-60 mph: 3.0 seconds
  • Top Speed: 193 mph
  • Transmission: 7-speed PDK
  • Weight: 3153 lbs

TECHNICAL DETAILS
